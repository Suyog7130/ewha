Gravitational Waves Data Analysis Tutorial

Welcome! This is a 3-part hands-on tutorial series on computational methods for gravitational waves data analysis. We will explore waveform implementations, ssh access to remote clusters, and introduction to neural networks using interactive Jupyter notebooks and markdown guides. All notebooks will be run live via RISE slides for an engaging session.

Tutorial Parts

  • Part 0: Set-up Colab, Jupyter, Conda
  • Part 1: Native Implementations of GW Approximants
  • Part 2: Working with Remote Computing Clusters
  • Part 3: End-to-end CNN for MNIST with keras・sklearn・TF・PyTorch
